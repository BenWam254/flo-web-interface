# SaaS Web Interface

A Minimalist frontend layout demonstrating precise typography hierarchies and fluid interface scaling.

---

## Key Features

- **Typography hierarchies** — Deliberate type scale across headings, subheadings, and body copy, with spacing and weight used as primary visual differentiators
- **Fluid viewport scaling** — Layout adapts consistently across mobile, tablet, and desktop breakpoints without reliance on third-party grid systems
- **Clean document flow** — Semantic HTML structure with a logical source order that holds up independent of styling
- **Lightweight interactive feedback** — Subtle state changes on interactive elements using vanilla JavaScript, with no external dependency overhead

---

## Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 |
| Behaviour | JavaScript (ES6+) |

---

## The Process

The core challenge was achieving a visually engaging interface through whitespace and typographic scale alone, without leaning on UI frameworks or component libraries as a crutch. Every spacing decision and type choice had to carry its own weight, since there was nothing else to fill the gap. The result is a layout that stays structurally clean precisely because the constraints were tight from the start.

---

## How to Run

Clone the repository and open the entry point directly in a browser — no build step required.

```bash
git clone https://github.com/Benwam/flo-web-interface.git
cd flo-web-interface
open index.html        # macOS
# or
xdg-open index.html   # Linux
# or open index.html manually in your browser of choice
```
